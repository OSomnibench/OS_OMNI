\section{Related Work}

\noindent
\textbf{Benchmarks for GUI Agents.} 
GUI-agent benchmarks have progressed from static web navigation to executable environments. Mind2Web, WebArena, VisualWebArena, WebVoyager, and WorkArena evaluate browser or enterprise web workflows \citep{deng2023mind2web,zhou2023webarena,koh2024visualwebarena,he2024webvoyager,drouin2024workarena}; AndroidWorld focuses on mobile-device interaction \citep{rawles2024androidworld}; and OSWorld, WindowsAgentArena, WorldGUI, macOSWorld, OS-MAP, and MMBench-GUI study desktop GUI automation \citep{xie2024osworld,bonatti2024windows,zhao2025worldgui,yang2025macosworld,chen2025map,wang2025mmbench}. These benchmarks establish the value of executable evaluation, but most still emphasize one platform family, one dominant interface assumption, or benchmark-specific runtime protocols.

\noindent
\textbf{Vision-language models for GUI Agents.} 
Recent CUA agents specialized for screen understanding and long-horizon interaction. AppAgent and MobileAgent study smartphone control, while SeeClick, ShowUI, and OmniParser improve visual grounding and screen parsing for GUI operation \citep{zhang2025appagent,wang2024mobile,cheng2024seeclick,lin2025showui,lu2024omniparser}. Agent~S explores open-ended desktop operation with explicit planning and recovery \citep{agashe2024agent}. More recently, OpenCUA and ScaleCUA develop open computer-using agents and training data for cross-platform control \citep{wang2025opencua,liu2025scalecua}. These systems show rapid progress, but their behavior remains sensitive to platform conventions and action interfaces, motivating unified cross-platform evaluation in \dataset.
